## 📋 PDF 转换方法

### 方法 1：使用 VS Code

1. 安装 `Markdown PDF` 扩展
2. 右键点击 `.md` 文件
3. 选择 `Markdown PDF: Export (pdf)`

### 方法 2：使用在线工具

- https://markdowntopdf.com/
- https://www.markdown.topdf.com/

### 方法 3：使用命令行（如有 pandoc）

```bash
pandoc 设计方案说明书.md -o 设计方案说明书.pdf
pandoc 检查清单/飞前检查清单.md -o 检查清单.pdf
pandoc 检查清单/检查流程图.md -o 检查流程图.pdf
```
